# Basketball CRUD API & Dashboard

A complete basketball management system consisting of:

- **REST API Backend**: Full CRUD operations for basketball league data (clubs, schedules, team stats, player stats)
- **React 19 Dashboard**: Modern TypeScript frontend with glassmorphism design for data management
- **MongoDB Integration**: Uses `bb-db` database with Mongoose ODM
- **OpenAPI Documentation**: Auto-generated API docs available at `/api-docs`

## Core Entities
- **Clubs**: Teams with players and coaches
- **Schedules**: Game schedules with locations, dates, and scores
- **Team Stats**: Win/loss records and points for clubs
- **Player Stats**: Individual player performance metrics

## Access Points
- API Server: http://localhost:3000
- API Documentation: http://localhost:3000/api-docs  
- Dashboard: http://localhost:3001
